# EvoHunt Artifacts

This repository will host the public artifacts for the EvoHunt paper:

**Transferable Self-Evolving Playbooks for Agentic Security Auditing**

We are currently organizing the release materials. The artifact package will include the materials needed to inspect the benchmark, scoring records, playbooks, and released traces at an appropriate disclosure level.

## RoadMap

We plan to release the EvoHunt artifacts in three phases.

### Phase 1: Evaluation artifacts (In Progress)

To support public verifiability and reproducibility, we will release:

1. All datasets, including the training set and testing set.
2. The execution results of each evaluated model on the testing set.
3. The judgment records for the execution results.

### Phase 2: Project source code (In Progress)

We will release the project source code, including:

1. The complete training pipeline.
2. The design and implementation of EvoHunt, the evaluator agent, and the reviser agent.

### Phase 3: Training playbook repositories (In Progress)

We will release the GPT and GLM training playbook repositories, including:

1. The complete commit history.
2. The movement history of the `playbook-best` and `playbook-candidate` branch pointers.
3. The final version of the playbooks used before paper submission.

Project website: <https://evohunt-project.github.io/>
